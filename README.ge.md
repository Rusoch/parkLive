Parklive рЃљрЃарЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЌрЃЉрЃўрЃџрЃўрЃАрЃћрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃърЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЊрЃљ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЏрЃЮрЃАрЃљрЃФрЃћрЃЉрЃюрЃљрЃЊ. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃЌ рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃАрЃљрЃърЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃАрЃўрЃЋрЃарЃфрЃћ, рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃў, рЃбрЃљрЃарЃўрЃцрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃљрЃљрЃЌрЃћрЃЉрЃў. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃАрЃљрЃърЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃФрЃћрЃЉрЃюрЃљ, рЃАрЃљрЃДрЃЋрЃљрЃарЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃљрЃарЃЕрЃћрЃБрЃџ рЃАрЃљрЃърЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃћрЃџ рЃљрЃЊрЃњрЃўрЃџрЃљрЃЏрЃЊрЃћ.

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:

- рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃарЃБрЃЎрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃАрЃљрЃърЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃЌ
- рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃАрЃљрЃърЃљрЃарЃЎрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃљ
- рЃФрЃћрЃЉрЃюрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃАрЃљрЃФрЃћрЃЉрЃюрЃљрЃЊ
- рЃАрЃљрЃДрЃЋрЃљрЃарЃћрЃџрЃў рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ
- рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ» рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ
- рЃЏрЃБрЃЦрЃў/рЃюрЃљрЃЌрЃћрЃџрЃў рЃЌрЃћрЃЏрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ
- рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ (рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў рЃЊрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџрЃў)
- Progressive Web App (PWA) рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
